## ✨ Возможности

- 💬 **Полноценный AI-чат** на основе Google Gemini
- 🎨 **Светлая и тёмная темы** с кастомным цветом акцента
- ⚙️ **Гибкие настройки** — шрифт, ширина, отступы
- 👤 **Профиль пользователя** — AI знает ваше имя, возраст, город
- 💾 **Сохранение истории** — все чаты остаются после закрытия
- 📎 **Отправка файлов** — фото, документы, код
- ⚡ **Стриминг ответов** в реальном времени
- 🎭 **Кастомные аватарки** для AI и пользователя
- 👍 **Лайки/дизлайки** сообщений
- 📋 **Копирование** одной кнопкой

## 📥 Установка

1. Скачай **MortiGTS.exe** из раздела [Releases](../../releases/latest)
2. Запусти файл двойным кликом
3. Заполни короткую анкету (имя, возраст и т.д.)
4. Введи свой API ключ от Gemini

## 🔑 Где взять API ключ

Бесплатный ключ от Google можно получить здесь:  
🔗 https://aistudio.google.com/app/apikey

## ⚠️ Системные требования

- **ОС:** Windows 10 / 11 (64-bit)
- **Зависимость:** [Microsoft Edge WebView2](https://developer.microsoft.com/microsoft-edge/webview2/) (обычно уже установлен)
- **Интернет:** Обязателен для работы AI
