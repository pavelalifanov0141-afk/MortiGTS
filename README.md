# 🚀 MortiGTS

Умный AI чат-бот на основе Google Gemini API. Создан командой **ClomeSquad**.

## ✨ Возможности

- 💬 Полноценный AI-чат на основе Gemini
- 🎨 Светлая и тёмная темы
- ⚙️ Гибкие настройки (цвет, шрифт, ширина)
- 👤 Профиль пользователя
- 💾 Сохранение истории чатов
- 📎 Отправка файлов и фото
- 🔥 Стриминг ответов в реальном времени

## 📥 Установка

1. Скачай **MortiGTS.exe** из раздела [Releases](../../releases)
2. Запусти файл
3. При первом запуске введи свой API ключ от Gemini

## 🔑 Где взять API ключ

Бесплатный ключ можно получить здесь: https://aistudio.google.com/app/apikey

## ⚠️ Требования

- Windows 10 / 11 (64-bit)
- [Microsoft Edge WebView2](https://developer.microsoft.com/microsoft-edge/webview2/) (обычно уже установлен)
